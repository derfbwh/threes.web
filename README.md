# threes.web

**threes.web** is a polished, browser-based tribute to the classic indie game *Threes!*. Developed as a personal project, it aims to capture the strategic depth and tactile feel of the original masterpiece while utilizing modern web technologies.

## ‚ú® Features
- **Authentic Mechanics:** Implements the "one-step" sliding movement and specific merging rules (1+2=3, 3+3=6, etc.).
- **Balanced Deck System:** Uses a non-random "bag" spawning algorithm to ensure fair play and strategic planning.
- **Late-Game Bonus Tiles:** Includes the "+" indicator and high-value bonus tile spawns once the board reaches elite levels.
- **Next Tile Indicator:** A visual preview of the upcoming card to allow for forward-thinking strategy.
- **Dark Mode:** A persistent, eye-friendly theme that saves to local storage.

- **HTML5:** Semantic structure.

- **Font Awesome:** Social integration.
